## 介绍

**这是一个基于electron写的app,你可以在linux/mac/win上使用，专为NCST的学生设计。**

你可能感兴趣的功能：

- 无需每次打开浏览器登陆校园网
- 软件只需设置时打开，设置完成即可关闭，不占用后台
- 支持学号，密码，验证码，服务选择(校园网/联通/电信/移动/教师组)
